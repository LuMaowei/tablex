# react-base-datagrid

[![NPM version](https://img.shields.io/npm/v/react-base-datagrid.svg?style=flat)](https://npmjs.org/package/react-base-datagrid)

基于 [react-window](https://github.com/bvaughn/react-window) 的表格组件

此组件为基础表格组件，只包含基础渲染部分，更多功能可查看 [Tablex](https://github.com/NexxLuo/tablex)

## Install

```powershell
yarn add tablex
```

```javascript
import React, { Component } from "react";
import Table, { flatten, unflatten } from "react-base-datagrid";

class Demo extends Component {
  state = {
    data: [],
    columns: []
  };

  render() {
    return (
      <Table rowKey="id" columns={this.state.columns} data={this.state.data} />
    );
  }
}
```

## 特性

1. 虚拟加载
2. 自适应宽、高
3. 无限级表头
4. 支持树形数据
5. 仿 antd table 样式
6. 兼容 ant table api
7. 列冻结
8. 列宽拖动
