# Tablex

基于<a href="https://github.com/bvaughn/react-window">react-window</a>的表格组件

<a href="https://nexxluo.github.io/tablex/src-table">Demo</a>

## 特性

1.大数据量虚拟加载

2.自适应宽、高

3.无限级表头

4.支持树形数据

5.仿 antd table 样式

6.兼容 ant table api
